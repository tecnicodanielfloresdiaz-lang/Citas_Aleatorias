# ✍️ Generador de Citas Aleatorias

> Una aplicación web sencilla que muestra frases inspiradoras de forma aleatoria al interactuar con la interfaz.

[🔗 Ver Demo en Vivo](https://tecnicodanielfloresdiaz-lang.github.io/Citas_Aleatorias/)
---

### 📝 Descripción
Este proyecto es una herramienta minimalista diseñada para practicar los fundamentos del desarrollo web. Utiliza un **arreglo (array)** de objetos en JavaScript para almacenar las citas y las muestra dinámicamente cada vez que el usuario hace clic en el botón.

### ✨ Características
- ✅ **Generación Instantánea:** Obtiene una nueva frase sin recargar la página.
- ✅ **Diseño Responsivo:** Adaptado para visualizarse correctamente en móviles y computadoras.
- ✅ **Interactividad:** Uso de eventos de escucha (EventListeners) para mejorar la experiencia de usuario.

### 🛠️ Tecnologías Utilizadas
- **HTML5:** Estructura semántica del generador.
- **CSS3:** Estilizado moderno con Flexbox y transiciones suaves.
- **JavaScript (Vanilla):** Lógica para la selección aleatoria mediante `Math.random()` y actualización del DOM.
